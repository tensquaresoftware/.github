Default **community health files** for [Ten Square Software](https://github.com/tensquaresoftware) open source projects.

GitHub automatically applies these defaults to every **public** repository on the account that does not define its own local copy. Individual repos can override any file if needed.

## Projects

| Project | Description | Status |
|---------|-------------|--------|
| [Matrix-Control](https://github.com/tensquaresoftware/matrix-control) | Cross-platform SysEx MIDI editor for the Oberheim Matrix-1000 (JUCE 8 plugin) | Public |

## Supporting the work

Ten Square Software projects are built in my spare time, alongside my day job as an instructional designer. [Cursor](https://cursor.com) has been a key part of my workflow for building tools like **Luthier**, **Matrix-Control**, **Sparkay**, and **Program Changer**.

If you find these projects useful, consider [sponsoring on GitHub](https://github.com/sponsors/tensquaresoftware) — every contribution helps cover tooling costs and keeps development going.

> 💛 Thank you to everyone who supports open source audio tooling!
